এই অ্যাপের মাধ্যমে আপনি র‍্যান্ডম ইউজারনেম ও শক্তিশালী পাসওয়ার্ড তৈরি করতে পারবেন এবং সেগুলো দিয়ে জিমেইল একাউন্ট খুলে এক্সেল আকারে সংরক্ষণ করতে পারবেন।

অ্যাপ ফিচারসমূহ

র‍্যান্ডম ইউজারনেম ও শক্তিশালী পাসওয়ার্ড জেনারেট করা হবে।

এই অ্যাপ থেকে যে জিমেইল দেওয়া হবে সেটা এখন পর্যন্ত exist করে না।

তৈরি করা ইউজারনেম ও পাসওয়ার্ড দিয়ে জিমেইল খুলে এক্সেল ফাইল আকারে ডাউনলোড করতে পারবেন।

API টোকেন ব্যবহার করে ম্যানুয়ালি আইডি জেনারেট করা যাবে।

গুরুত্বপূর্ণ তথ্য

  • একসাথে অনেক আইডি জেনারেট না করার পরামর্শ, কারণ হোস্টিং ও ডাটাবেজ ফ্রি ট্রায়াল।
  • একাউন্ট পাসওয়ার্ড রিকভারি নেই, তাই ভালোভাবে সংরক্ষণ করুন।

উপরের কমান্ড রান করলে আপনার একাউন্টে ২০টি বাল্ক জিমেইল তৈরি হবে। আপনার API টোকেন প্রাপ্ত করতে আপনার প্রোফাইল পেজ দেখুন।

কিভাবে ব্যবহার করবেন

1

রেজিস্ট্রেশন করুন

সহজ রেজিস্ট্রেশন প্রক্রিয়া সম্পন্ন করে আপনার একাউন্ট তৈরি করুন।

2

আপনার API টোকেন প্রাপ্ত করুন

আপনার প্রোফাইল থেকে API টোকেন সংগ্রহ করুন।

3

কমান্ডটি কপি করুন

উপরের কমান্ডটি কপি করে আপনার টার্মিনালে রান করুন (API_Token অংশটি আপনার টোকেন দিয়ে রিপ্লেস করুন)।

4

ডাটা এক্সপোর্ট করুন

সমস্ত একাউন্ট তথ্য এক্সেল ফাইল হিসেবে ডাউনলোড করুন।
